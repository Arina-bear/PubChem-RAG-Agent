# 🧪PubChem-RAG-Agent
[![Status](https://img.shields.io/badge/Status-In%20Development-orange)](https://github.com/Hominizm112/hakaton) <!-- Замените ссылку и статус при необходимости -->

Агент на основе RAG (Retrieval-Augmented Generation), который по текстовому запросу находит молекулы в PubChem и возвращает их свойства.
Сделать агента, который получает информацию из PubChem по описанию свойств молекул в виде текста и возвращает пользователю

## 📦 Установка и запуск

Самый быстрый путь сейчас:

1. Создать `backend/.env` из [backend/.env.example](backend/.env.example)
2. Вставить туда ключ `MODAL_GLM_API_KEY` или `OPENAI_API_KEY`
3. Запустить проект одной командой:

```bash
./scripts/dev.sh
```

После этого:

- frontend: `http://localhost:3000`
- backend: `http://127.0.0.1:8000`

Подробная инструкция по API, ключам, `curl`, JavaScript и Python примерам лежит в [readme-api.md](readme-api.md).
