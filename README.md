# 🧪PubChem-RAG-Agent
[![Status](https://img.shields.io/badge/Status-In%20Development-orange)](https://github.com/Hominizm112/hakaton) <!-- Замените ссылку и статус при необходимости -->

Агент на основе RAG (Retrieval-Augmented Generation), который по текстовому запросу находит молекулы в PubChem и возвращает их свойства.
Сделать агента, который получает информацию из PubChem по описанию свойств молекул в виде текста и возвращает пользователю
